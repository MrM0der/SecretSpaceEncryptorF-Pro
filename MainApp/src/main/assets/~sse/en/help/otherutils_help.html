Clipboard Cleaner

  • Erase content of the system clipboard.
  • The Clipboard Cleaner can work properly only with the system clipboard. If your phone uses a clipboard manager (for example Clip Tray on LG phones, keyboard clipboard on Samsung phones, etc.) be aware of this: The clipboard managers have own cache for "historic" system clipboard states and the SSE or any other app has no rights to delete or modify this data (it's in an internal data space of the clipboard manager). It can only try to "push out" the data by a multiple use of a cleaning mechanism.
