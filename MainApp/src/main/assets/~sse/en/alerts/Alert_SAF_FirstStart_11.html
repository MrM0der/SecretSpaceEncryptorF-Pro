SAF Mode
  • Use this mode only for locations that are not accessible (or writable) in the standard way (using button).
  • SAF mode provides significantly lower performance and has some limitations.

Notes & Limitations

  • SAF API provides much lower performance than the standard direct file access mode (especially with dirs that contain a lot of files/subdirs).
  • Home/Custom directories cannot be set in SAF; however, the last-picked SAF URI is automatically saved: Home Button → Go to Last Picked SAF Dir.
  • The wipe feature does not work. If option “Wipe after Encryption” is checked in the app settings, file(s) will be deleted only.
  • In general, external SD cards are usually much less reliable (especially some combinations of devices/SD cards) than an internal memory. Also, be sure to always safely unmount SD cards (and other external storages) before removing.
